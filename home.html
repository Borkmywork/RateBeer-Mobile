RateBeer Mobile for Android Project

The RateBeer Mobile for Android Project aims to provide a userfriendly and fast mobile Ratebeer experience.

The project is released under the GPL license and it is intended to be a collaborative effort.

RateBeer Mobile for Android is a third party android application, for interacting with the ratebeer.com site. The project is not created by, or run by ratebeer.com staff. So any complaints or suggestions should not be directed to the ratebeer.com site.
